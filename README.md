# skillquest

![Status: Work in Progress](https://img.shields.io/badge/Status-Work%20in%20Progress-yellow)

A to-do list for learners. Create quests, break them into subquests, and complete tasks to level up your skills.

## Motivation

This project was developed as part of the <span><a href="https://www.theodinproject.com/paths/full-stack-javascript">Odin Project Full Stack JavaScript</a></span> curriculum, with the following goals:

- Strengthen core JavaScript skills using modern ES6 modules
- Practice organizing code into clean, reusable components
- Learn how to bundle and manage assets using Webpack
- Build a structured application with clear state management
- Apply clean coding principles and modular architecture
